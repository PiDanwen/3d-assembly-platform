# 2026-05-06 工作记录

## 3D模型搭建配置平台

- 创建文件：`3d-assembly-platform.html`
- 技术栈：Three.js (ESM via jsdelivr CDN)、原生HTML/CSS/JS
- 核心功能：
  1. **3D模型上传解析**：支持GLB/GLTF/OBJ格式上传，自动解析零件树
  2. **演示模型**：内置12个零件的演示装配模型，可立即加载体验
  3. **零件列表**：左侧面板，支持搜索、配置状态指示（绿/黄/灰点）
  4. **3D视口**：Three.js渲染，支持旋转/平移/缩放、点击选中零件、线框模式、网格切换、隔离模式
  5. **零件属性**：位置XYZ、旋转欧拉角、颜色、材质编辑
  6. **安装方向配置**：6向选择（±X/±Y/±Z），带方向箭头可视化，防穿模方向冲突检测
  7. **碰撞检测**：自动分析零件距离与方向冲突，生成报告
  8. **装配步骤面板**：按步骤顺序展示，支持自动重排
  9. **搭建预览动画**：按安装方向逐件动画装入，支持播放/暂停/进度拖拽
  10. **导出配置**：生成assembly_config.json
- 深色主题设计，类专业级3D软件UI

## 追加功能：360° 旋转预览

- 工具栏新增「360° 旋转预览」按钮，选中零件后点击激活
- 相机围绕零件包围盒中心做轨道环绕旋转（camera orbit，非mesh自转）
- 右下角控制面板：速度滑块（0.2×~3.0×）、旋转轴切换（Y/X/Z）、暂停/继续、复位
- 暂停时自动启用 OrbitControls，继续时无缝同步相机位置再接管
- 切换零件时自动重新聚焦到新零件

## 追加功能：零件360°预览小窗口

- 在零件属性面板下方添加独立小型3D预览窗口（260×180px）
- 独立 Three.js 渲染器（miniRenderer/miniScene/miniCamera）
- **选中零件后自动开始预览**，无需手动点击按钮
- 零件居中显示，自动缩放适配
- 顶部速度滑块控制旋转速度（0.2×~3.0×）
- 左下角显示零件名称 + 旋转图标
- 重置/切换零件时自动更新预览内容

## 修改：手动步骤预览 + 位置保护

- **手动步骤预览**：移除自动播放，改为手动「上一步/下一步」按钮控制
- **位置保护**：添加 `saveOriginalPositions()` 和 `restoreOriginalPositions()` 函数
- 预览过程中零件位置始终保持原始位置不变
- 「退出预览」按钮恢复完整模型显示
- 「重置」按钮从第一步重新开始
- 进度条点击可跳转到任意步骤
- 装配步骤列表点击可跳转到对应步骤预览
